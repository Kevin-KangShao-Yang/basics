- [ ] img标签的属性分别是

  ```
  img用来加载图片
  src:图片的路径,保证后缀名准确
  alt:当图片加载失败的时候,显示的描述文本
  title:当鼠标悬停的时候,显示的描述文本
  width/height:控制图片的大小宽高----只控制一边,另外一边会等比例缩放,这样就不会变形
  ```
