css第一天重点落地笔记

1.input控件学习
	类型:
		button---普通按钮
		submit---提交按钮
		reset-----重置按钮----必须配合form标签
		image----图片按钮----src为图片路径
	拓展标签
		<button>新按钮</button>

2.label标签绑定
	将文字与input控件绑定到一起,实现点击其中任何一个就相当于点击了另外一个
	1.利用for属性找到id对应的元素
	2.用label嵌套目标元素,for属性必须清除

3.行内式css
	写在标签内部
	style
	只能影响到当前一个标签

4.内嵌式css
	写在head标签中
	<style></style>
	影响当前的一个页面

5.外链式css
	单独存在的css文件(后缀为.css)
	<link>来导入css文件
	影响当前一个网站

6.color属性
	英文单词赋值,范围太小
	rgb表示法,0-255
	十六进制,00-ff
	工作中,CV就可以了

7.字体大小
	以px作为单位是最常见的
	拓展tm单位,取决于当前的默认字体大小
	谷歌浏览器默认字体大小16px
	最小字体大小12px

8.字体库
	默认字体是微软雅黑

9.字体加粗
	normal:正常---400
	bold:加粗------700

10.字体倾斜
	normal:正常
	italic:倾斜

11.font连写
	多个属性用空格隔开
	后面的两个属性不能省略,不能调换位置
	必须先写字体大小,再写字体库
