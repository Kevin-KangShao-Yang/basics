- [ ] input常见的类型以及属性

  ```
  input
  	类型
  		text-----文本框
  		password-密码框
  		radio----单选框
  		checkbox-多选框
  		button---普通按钮
  		submit---提交按钮
  		reset----重置按钮----form表单域配合使用
  		image----图片按钮----src属性图片路径
  		file-----文件域
  		date-----日历控件
  		color----调色板控件
  	属性
  		value-------默认值
  		placeholder-占位符
  		maxlength---最大字符数
  		checked-----单选/多选默认选中
  		name--------设置分组名
  		selected----下拉框默认选中
  	标签
  		<button>-----新版本按钮
  		<textarea>---文本域
  		<select>-----下拉框
  		<option>-----下拉选项
  		<form>-------表单域
  		<label>------绑定标签
  		
  ```

- [ ] 关于css的三种引入方式

  ```
  css是负责网页的美容
  1.行内式
  	写在标签的内部
  	style
  	当前一个标签
  2.内嵌式
  	写在head标签中
  	<style>
  	当前一个页面
  3.外链式
  	独立存在css文件
  	<link>导入文件
  	当前一个网站----工作中最常用的方式
  ```

- [ ] 关于字体四个属性设置

  ```
  1.font-size:控制字体大小
  	px/em
  	em是当前的默认字体大小,本身是在变动的
  	浏览器默认字体大小16px
  	浏览器最小字体大小12px
  2.font-family:控制字体库
  	备胎计划,提供多个字体,用逗号隔开
  	默认字体微软雅黑
  3.font-weight:控制字体是否加粗
  	normal:正常字体----400
  	bold:加粗字体------700
  4.font-style:控制字体是否倾斜
  	normal:正常字体
  	italic:倾斜字体
  ```

- [ ] 关于font字体的连写

  ```
  font: 加粗 倾斜 大小 字体库;
  1.前面两个可以调换位置,可以省略(加粗和倾斜)
  2.后面两个不能调换位置,也不能省略(必须先写字体大小,再写字体库)
  ```

- [ ] 三种文本颜色的设置方式

  ```
  color: 控制文本的颜色
  	1.英文单词赋值: 范围太小,建议练习的时候使用
  	2.rgb表示法:将颜色分为红绿蓝,0-255
  	3.十六进制:将颜色分为红绿蓝,00-ff-----工作中用得最多的
  	记住一点:我们只要复制黏贴即可
  ```
