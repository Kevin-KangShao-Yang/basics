1.层叠性
	当相同选择器(权重相同)作用在同一个元素身上,发生属性冲突,后面的属性会覆盖前面的,后来者居上.

2.继承性: 
	存在父子关系的元素,子元素会继承父元素的部分属性
	a标签不继承颜色
	h系列标签不继承字体大小

3.优先级的顺序:
	继承
	通配符
	标签
	类
	id
	行内式
	!important

4.盒子模型:
	边框:border
	内边距: padding
	外边距: margin

5.边框的属性:
	border-style:边框的样式----solid实线
	border-color:边框的颜色
	border-width:边框的宽度

6.边框的连写方式:
	border: 宽度 样式 颜色;
	border: 1px solid red;

7.圆角边框:
	border-radius: 半径;
