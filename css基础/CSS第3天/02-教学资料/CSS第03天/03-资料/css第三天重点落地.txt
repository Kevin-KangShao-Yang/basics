1.交集选择器
	选择器名之间没有任何链接符号
	找到一个元素,这个元素拥有所有的条件

2.什么时候使用交集
	任何情况都不要主动使用其他选择器
	第一选择是后代选择器
	如果后代没有办法控制元素,再配合其他选择器

3.并集选择器
	选择器之间用逗号隔开
	每个条件单独一行
	控制多个目标元素

4.a标签的伪类选择器
	link: 未被访问链接
	visited:被访问过的链接
	hover:鼠标悬停的链接:工作中最常用的
	active:点击瞬间的链接

5.块级元素:
	block
	独自占据一行
	可以设置宽高
	没有宽度的话,继承父元素的宽

6.行内元素:
	inline
	一行显示多个
	不能设置宽高,由内容决定宽高

7.行内块元素:
	inline-block
	一行可以显示多个
	可以设置宽高

8.模式转换:
	display: block/inline/inline-block
	先分析功能需要什么显示模式,然后在通过属性控制

9.背景颜色的透明程度
	rgba:0-1,背景透明
	opacity:0-1,全局透明

10.背景图片的设置:
	background-image:url("图片的路径")
	前提: 盒子本身必须要有宽高
	background-repeat:控制图片的平铺方式
	no-repeat:不平铺
	background-position:
		1.利用数值,先写x轴,再写y轴
			水平往右为正
			垂直往下为正
		2.利用方位名词:九种

11.背景的连写:
	background: 路径 是否平铺 位置 背景颜色;
